Description and Features

This module add a task manager with several possibilities like assign task to users, manage work flow and reminder the related users by e-mail.

Module Features :

  • PostIt - menu item : to open the PostIts (tasks) list / form view
    • Team: the team to who the task has been assigned to
    • Name : task Name
    • Tags
    • Assigned to : the assigned user(s)
    • Copy to : users will also receive the reminder email
    • Assigned by : the user who created the task
    • Client : the task related partner
    • Days : for reminder - week days
    • Priority
    • Date start : for reminder - start date
    • Date end : for reminder - end date
    • Recall Date
    • Duration
  • PostIt Tag - menu item : to open the Postits Tags list / form view
  • PostIt Team - menu item : to open the Postits Teams list / form view
