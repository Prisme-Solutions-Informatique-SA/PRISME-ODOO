Description and Features

Add a list view to display all suppliers invoices lines.

Module Features :

  • Invoicing tab - Vendors
    • Adds new "Supplier Invoice Line" menu item to open supplier invoices lines list view.

  • Fields inside the supplier accounting lines - Measure
    • Amount Signed : debit and credit in local currency (negative or positive amount).
    • Amount : amount in foreign currency.

The new menu item located inside Purchases tab.

The supplier invoice line's list view.
