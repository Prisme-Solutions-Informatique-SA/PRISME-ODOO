Description and Features

Add functionalities to significantly improve the management of S/N (Lots) with warranties and extras fields.

Module Features :

  • Warranties : add warranties management with menu items (Warranties, Warranty Types) on Warehouse/Traceability
    • Warranties - Form/List View
      • Description
      • Contacts
      • Informations: contains lots and state informations
      • Invoices : related to customer and supplier invoices
      • Dates
        • Start Date
        • End Date
        • Reminder Date : to inform user by email
      • Internal Notes
      • Notes for Contact
    • Warranty Types - Form/List View : used to create warranty types like HW warranty and SW warranty
  • Serial Numbers - Form View
    • Inventory Tab : informations about this lot
      • Description section
      • Customer/User section
      • Attachments section
      • Manufacturer section
      • Supplier section
      • Remarks section
      • End life date section
    • Warranties Tab : lists all related warranties
  • Serial Numbers - List View
    • Customer
    • Description
  • Creates a warranty management scheduler that sends mail to assigned contacts after reminder date
    • Go to Settings / Technical / Automation / Scheduled Action
    • Search for "Warranty Management Scheduler" and click "Activate" to send reminder emails

Installation

Prerequisites :

  • This module sends emails, hence it needs the outgoing mail server to be set up.

Steps :

  1. Click the install button and it's done

Configuration

  • To activate lots management, you have to activate "Track lots or serial numbers" in Settings / Configuration / Inventory / Traceability
