Description and Features

Add functionalities to improve the management of purchase orders

Module Features :

  • Purchase order
    • Form view: Since PO is confirmed then "Source Document" field could not anymore be displayed / modified. This module allows user to show "Source Document" with confirmed status and allows user to edit the content of this field.
