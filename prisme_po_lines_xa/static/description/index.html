Description and Features

Adds a list view to display all purchase orders lines.

Module Features :

  • Purchases - Purchases tab
    • Adds a new "Purchase lines" menu item.
    • Opens the purchase order lines' list view.
    • A field "Amount in local currency" to support multi-currency is added.

  • Purchase Lines
    • Contains a list, form and pivot view.
    • Displays purchase line informations.
