Description and Features

Adds a list view to display all customers invoices lines.

Module Features :

  • Invoicing tab - Customers
    • Adds new "Customer Invoices Lines" menu item to open customer invoices lines list view.
    • Adds "Signed Subtotal" field in the form view of the customer invoice line.

The new menu item located inside Sales tab.

The customer invoice line's list view.

The customer invoice line's form view.

Installation

Prerequisites :
  • Make sure that you have installed the "Invoicing Management" (accounting_invoicing) module first before installing this module.
Steps :
  1. Click the install button and it's done.
