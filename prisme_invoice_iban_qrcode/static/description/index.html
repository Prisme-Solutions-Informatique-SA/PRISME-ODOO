Description and Features

Adds report based on the original with QR-bill.

Create new copy of Invoices report with QR-bill payment part

  • Follows Swiss Implementation Guidelines QR-bill
  • Uses BVR reference if available in invoice

Add QR-bill tab to customer invoice.
Displays the QRCode if possible or the reason why the QRCode is not available

Frequently asked questions (FAQ)

Why is the payment part not present?

If the payment part is not present in the report, there must be a reason. On the client invoice view there is a tab named "QR-Invoice" which contains the QRCode or the reason why the QRCode is not present.

Why is there a new report in the vendor bill?

Unfortunately there is no way to add a report in the customer invoice without adding it to the vendor bill.
The vendor bill and the customer invoice are the same object and the report is linked to this object.
