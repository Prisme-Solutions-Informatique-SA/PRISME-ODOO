Description and Features

Adds a list view to display all sale orders lines.

Module Features :

  • Sales - Orders tab
    • Adds a new "Sales lines" menu item.
    • Opens the sale order lines' list view.

  • Sales Lines
    • Contains a list, form and pivot view.
    • Displays sale line informations.
    • A field "Amount in local currency" to support multi-currency is added.

New menu item located inside Orders tab.


Sale order lines' list view.




Sale order lines' form view. Total and subtotal amounts for the line are also displayed below the Analytic Tags field.

Installation

Prerequisites :
